# Contribution Guidelines

When contributing to SneakrVault, please follow these guidelines:

## How to Contribute

### Prerequisites

Start by checking if there isn't already an issue open with what you want to contribute with, if not start by [opening an issue](https://github.com/D3W10/SneakrVault/issues) describing the problem you would like to solve.

### Setup your environment locally

Follow the instructions in the [README.md](README.md) to setup your local environment.

### Implement your changes

You may fork this repository and make changes to the code on a separate branch. Once you're done, submit a pull request to the main branch.
